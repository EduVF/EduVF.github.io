MACHINE

a real-time battery-powered senseless selfish heart

is a piece written for drums and live electronics. The special part of it is that all the electronics are controlled using a pair of Joy-Con controllers from the Nintendo Switch game console. It works through Bluetooth, paired to a Python script that sends OSC messages to SuperCollider, the software that ultimately does the sound synthesis.

The score uses a combination of traditional writing for the drums, and spectrum-like graphics for the electronic parts. However, both performers are encouraged to experiment with the system and work together to improvise and create their own interpretation of the music.
